<h3 style="font-family: Georgia, serif; color: #3D2F20; font-size: 16px; margin: 24px 0 12px 0;">Key Features</h3>
<ul style="color: #5A4733; line-height: 1.8; padding-left: 20px; margin: 0 0 20px 0;">
<li>Warm amber/cream/gold/sage palette with Playfair Display serif font</li>
<li>Masonry memory wall with hover animations</li>
<li>Chronological timeline view with vertical connector</li>
<li>Long-form stories view</li>
<li>Ken Burns effect slideshow with keyboard controls (arrows, space, esc)</li>
<li>Invite system: shareable links + invite codes</li>
<li>Privacy controls: public, private, family-only</li>
<li>Reaction system (heart, candle, flower)</li>
<li>Photo upload to Supabase Storage</li>
<li>Full Supabase Auth (sign up, sign in, sign out)</li>
<li>Row Level Security on all tables</li>
<li>Responsive design throughout</li>
</ul>

<h3 style="font-family: Georgia, serif; color: #3D2F20; font-size: 16px; margin: 24px 0 12px 0;">Design Notes</h3>
<p style="color: #5A4733; line-height: 1.7; margin: 0 0 20px 0;">
No indigo. No AI slop. The palette is entirely warm tones -- amber (#F5A623), cream (#FFF9ED), gold (#D4A843), sage (#7A9466), and earth (#3D2F20). Typography uses Playfair Display for headings and Inter for body text. The aesthetic is nostalgic and gentle, like opening a treasured photo album.
</p>
